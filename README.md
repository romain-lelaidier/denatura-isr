# Uranium ISR scripts

### dynamic_placement.ipynb

Generates a wells geometry (hexagonal / organic) depending on the input uranium distribution and cell characteristic radius.

### scripts_generator.py

Generates and launches HYTEC simulation jobs with varying parameters.

### post_processing.py

Plots the results of a simulation.

### compare.py

If a bunch of simulations have been launched by scripts_generator.py with a varying parameter, plots some result indicators depending on this parameter.
